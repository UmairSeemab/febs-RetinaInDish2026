# Retina in a Dish 2026 Interactive Abstract Book

Retina in a Dish 2026 Interactive Abstract Book

## Contents

- Searchable program schedule
- Searchable poster abstract book
- Participant directory
- Sponsor section
- Original abstract files linked from each abstract page

## Update data

Edit JSON files in `data/`:

- `data/program.json`
- `data/abstracts.json`
- `data/participants.json`
- `data/sponsors.json`

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to repository Settings.
4. Open Pages.
5. Select branch `main` and root folder `/`.
6. Save.

Your app will be published as a GitHub Pages website.
